Office Hours

Push-to-talk intercom for teams. LAN or remote.

Download ZIP

Setup

  1. Unzip the download and open the folder
  2. Double-click Office Hours.command
  3. If macOS blocks it: go to System Settings > Privacy & Security and click Open Anyway
  4. The app installs everything it needs on first run
  1. Install Python first (check "Add to PATH" during install)
  2. Unzip the download and open the folder
  3. Double-click Office Hours.bat
  4. The app installs everything it needs on first run

When the app opens, enter your name. If you have an invite code, paste it in to join your team.
