How it works
This plugin connects to the Vox desktop app running on your computer. Make sure Vox is open before using these buttons.

Push to Talk — Hold to transmit, release to stop
Status Mode — Tap to cycle: Available → Busy → DND
Switch Team — Tap to cycle through teams
Select User — Tap to cycle through online users
Show Panel — Open/close the Vox panel window
